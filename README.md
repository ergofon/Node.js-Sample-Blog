Node.js sample blog
=============

A sample blog built with Node.js, Express, Mongoose (MongoDB), and Jade.

Setup
-------

	git clone git://github.com/trantorLiu/Node.js-Sample-Blog.git
	cd Node.js-Sample-Blog
	npm install
	node app.js

Then visit [http://localhost:3000].
